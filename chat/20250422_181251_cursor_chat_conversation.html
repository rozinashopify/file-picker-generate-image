Ah yes, you're absolutely right! According to the CSS-Tricks article, border-image-source (and by extension border-image) does not work with border-radius. The corners will remain sharp even if you set a border-radius on the element.

To achieve rounded corners with a gradient border, we'll need to use a different approach. Here are a couple of solutions:

  1. Using a pseudo-element with background-image:
.loader-rectangle {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  width: 100%;
  height: 100%;
  border-radius: 25px;
  box-sizing: border-box;
  background-color: transparent;
}

.loader-rectangle::before {
  content: '';
  position: absolute;
  top: -2px;  /* Match border width */
  right: -2px;
  bottom: -2px;
  left: -2px;
  background: linear-gradient(to bottom, 
    #FFD8B7 0%, 
    #F4C2FF 9%, 
    #D1BDFF 18%, 
    #A0CBFF 27%, 
    #A0CBFF 36%, 
    #D1BDFF 45%, 
    #F4C2FF 55%, 
    #FFD8B7 64%, 
    #FFD8B7 73%, 
    #F4C2FF 82%, 
    #D1BDFF 91%, 
    #A0CBFF 100%);
  border-radius: 25px;
  z-index: -1;
}
  1. Or using a mask with background-image:
.loader-rectangle {
  position: absolute;
